## 🚀 About Me

I'm a **Minecraft plugin developer** focused on building custom systems for **PaperMC servers**.

I like creating plugins with clean configuration, useful commands, GUI systems, economy integration, leaderboards, and unique gameplay mechanics.

Currently working on:

- Quest systems
- Adaptive boss AI
- GUI leaderboard systems
- Server utilities
- Survival, dungeon, and creative server features

---

## 🧩 Featured Projects

### QuestSP

A lightweight quiz and quest plugin for Minecraft Paper servers.

**Features:**

- Configurable quiz questions
- Vault economy rewards
- Permanent leaderboard
- GUI leaderboard
- Admin commands
- Paper 1.21.x support

### BossAI

An adaptive boss system where bosses can learn from player combat behavior.

**Features:**

- Boss level system
- Player combat memory
- Anti-melee tactics
- Anti-ranged tactics
- Anti-shield behavior
- Arena level requirements
